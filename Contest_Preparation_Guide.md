#🏆 Contest Preparation & Mental Models

> **Contests are 40% technical skill and 60% mental preparation. Master both to perform at your peak when it matters most.**

---

##🧠 Contest Mental Models

### 1. The Iceberg Model
**Visible (40%):** Problem-solving, coding, algorithms
**Hidden (60%):** Mental state, stress management, decision-making

**Preparation Focus:**
- Technical: Practice, review, study
- Mental: Confidence, stress inoculation, mindset

### 2. Energy Management Framework
**Cognitive Energy:** Mental focus and clarity
**Emotional Energy:** Stress levels and motivation
**Physical Energy:** Rest, nutrition, body state

**Optimization Strategy:**
- Peak cognitive energy for hardest problems
- Manage emotional energy through breathing/positivity
- Maintain physical energy with proper rest/nutrition

### 3. Decision-Making Under Pressure
**Problem Selection:** Read all problems first, start with easiest
**Time Allocation:** 20-25% per problem for 4-problem contests
**Risk Assessment:** When to persist vs when to move on
**Quality Control:** When to double-check vs when to submit

## 🎯 Contest Strategy Framework

### Phase 1: Initial Assessment (First 15 minutes)
1. **Read ALL problems** (don't start coding yet)
2. **Rate difficulty** for each problem (1-5 scale)
3. **Identify easiest problem** to build momentum
4. **Plan time allocation** based on your strengths
5. **Set breakpoints** for when to move between problems

### Phase 2: Execution (Middle Phase)
**Problem Selection Order:**
1. **Easiest problem first** (build confidence)
2. **Second easiest** (maintain momentum)
3. **Your strongest tag** (maximize points)
4. **Remaining problems** (challenge phase)

**Time Management:**
- **First 2 problems:** 30-40 minutes total
- **Strong tag problem:** 45-60 minutes
- **Remaining time:** 30-45 minutes for last problem(s)

### Phase 3: Optimization (Final 30 minutes)
- **Review all solutions** for obvious errors
- **Check edge cases** you might have missed
- **Optimize time-consuming solutions** if possible
- **Make educated guesses** on unsolved problems if time is running out

## Game Strategies

### Stress Management Techniques:
**Breathing:** 4-7-8 technique (inhale 4, hold 7, exhale 8)
**Positive Self-Talk:** Replace "I can't" with "I'm working on it"
**Focus Anchoring:** Concentrate on one problem at a time
**Perspective Reset:** Remember everyone else is nervous too

### Confidence Maintenance:
**Success Anchoring:** Recall 3 recent successful problem-solving moments
**Progress Tracking:** Note small wins during contest
**Momentum Building:** Start with problems you know you can solve
**Recovery Strategy:** If stuck, take 2-minute break, then return

### Decision Fatigue Management:
**Pre-decided Rules:**
- If stuck for 15 minutes, move to next problem
- If 2+ wrong answers, re-read problem carefully
- If time < 30 minutes, focus on debugging existing solutions
- If unsure about approach, try brute force first

3. **Consistency Building:** Regular participation builds skills
4. **Stress as Fuel:** Channel nervousness into focused energy

##📈 Progression Path

### Beginner Level (Rating 800-1200):
**Goals:**
- Solve 1-2 problems consistently
- Build basic problem recognition
- Develop contest rhythm
- Learn time management

**Strategy:**
- Focus on A/B problems first
- Build confidence with easier problems
- Learn to read problems carefully
- Practice basic implementation speed

### Intermediate Level (Rating 1200-1600):
**Goals:**
- Solve 2-3 problems consistently
- Handle medium-difficulty problems
- Develop systematic approach
- Build contest endurance

**Strategy:**
- Master core algorithms
- Practice common contest patterns
- Work on problem selection skills
- Develop debugging efficiency

### Advanced Level (Rating 1600+):
**Goals:**
- Solve 3-4 problems consistently
- Handle hard problems regularly
- Compete for high rankings
- Mentor others

**Strategy:**
- Master advanced algorithms
- Practice under extreme time pressure
- Develop unique problem-solving approaches
- Build competition experience
